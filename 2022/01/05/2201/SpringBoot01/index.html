SpringBoot概述

Spring Boot 是所有基于 Spring Framework 5.0 开发的项目。Spring Boot 的设计是为了让你尽可能快的跑起来 Spring 应用程序，并且尽可能减少你的配置文件。

设计目的： 用来简化 Spring 应用的初始搭建以及开发过程。
